# 3.2.9 transfer

>说明: 用直观的方式在两栏中移动元素，完成选择行为

##### 使用及参数说明:
```html
<div id="transfer" class="xui-flex xui-transfer"></div>
```

```js
new xui.transfer({
	id: string,
	prevList: array,
	lastList: array,
	fn(data){
		{
			leftList: [
				{
					value: "javascript",
					disabled: "false"
				}
			],
			rightList: [
				{
					value: "todo",
					disabled: "false"
				}
			]
		}
	}
});
```
* `id`表示穿梭框容器,**必传**
* `prevList`表示穿梭框左边显示的数据(原始数据)
* `lastList`表示穿梭框右边显示的数据(目标数据)
* `fn`表示选择后的回调,返回一个对象包含原始数据和目标数据
* 没有你想要的功能? 快联系我: `xumeng0611@gmail.com`添加吧
