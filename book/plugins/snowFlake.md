# 3.2.3 snowFlake

>说明: 希望给页面上某个元素添加雪花飘洒的效果

##### 使用及参数说明:
```js
xui.snowFlake({
	id: string,
	direction: string
})
```
* `id`表示雪花容器,**必传**
* `direction`表示雪花的飘洒方向, 默认向右, 可以设置为`left`
* 需要注意的是, 建议给你容器一个宽高 ~_~
* 没有你想要的功能? 快联系我: `xumeng0611@gmail.com`添加吧
